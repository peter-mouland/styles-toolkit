Breakpoints

All pages should be built for mobile (xs) devices and then improved for larger screen. The following breakpoints are the recommended screen-sizes for which the page should work at.

xs
48em (320px)
sm
48em (768px)
md
64em (1024px)
lg
75em (1200px)

To use the breakpoints we recommend to use the the breakpoint mixin

Grid

Each page should use the n-container-fluid class name. This container should consist of sections of content using the n-row class name. These elements should not normally be nested.

Each row should then consist of a maximum of 12 columns.

Example 
<div class='n-container-fluid'>
    <div class='n-row'>
      <div class='n-col-xs-10'>wide-column</div>
      <div class='n-col-xs-2'>thin-column</div>
    </div>
  </div>

For full Grid class names and usage help, see the grid demo page.
